------class
class에 대한 개념을 알아보았는데, 사실 class는 c언어의 구조체와 매우 흡사해 보였다.
일단 객체지향 프로그래밍 언어란 프로그램을 다수의 객체로 만들고 이들끼리 서로 상호작용하게 하는 것인데, 
구조체와 class의 차이점은 같은 클래스 object를 여러 개의 변수에 할당하고
값을 변경시키면 할당한 모두 영향을 받지만 구조체는 그렇지 않다.
python에서 class를 어떻게 사용하는지 알아보자.

class hello:
    def __init__(self,name):
        self.name = name

    def hello(self):
        print("안녕 "+self.name+" 만나서 반가워")

a = hello("철수 ")
a.hello()

이런식으로 사용한다. 여기서 a는 인스턴스 혹은 객체라고 불린다.
그냥 일반적으로 함수를 구성할때와는 다르게 class내부에 있는 모든 함수의 첫번째 매개변수에는 self가 들어간다.
self란 인스턴스 자신이 들어가게 된다.
즉 함수 내부에서 self.name 에 값을 넣는 다는 것은 a.name 에 값이 들어간다는 것이다.
이런식으로 class를 이용할수 있다.
