오늘은 명령어 세트에 대해서 공부하였다.  
명령어 사이클에 대한 내용에서도 살펴 보았는데 꽤나 복잡해 보이기 때문에 추가적으로 알아 봐야 될 것 같다.  

명령어 세트

CPU가 실행할 명령어의 집합

연산코드 + 피연산자 로 이루어져 있다.
- 연산코드 : 실행할 연산
연산, 제어, 데이터 전달, 입출력 기능을 가졌다.

- 피연산자 : 필요한 데이터, 저장 위치
주소, 숫자/문자, 논리 데이터 등을 저장한다.

CPU가 프로그램을 실행하는 과정

주기억장치에서 명령어를 순차적으로 인출  => 해독 => 실행
